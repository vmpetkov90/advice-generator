Styled with **TailwindCSS** and powered by **JavaScript**, this project delivers a smooth and interactive experience.

## 🚀 Features

- Fetches random advice from the Advice Slip API  
- Smooth loading animation on the dice button  
- Fully responsive layout using TailwindCSS  
- Clean, centered card UI  
- Error handling for failed API requests  
- Auto-loads advice on page load  

---

## 🛠️ Technologies Used

- **HTML5**
- **TailwindCSS**
- **JavaScript (ES6)**
- **Advice Slip API**

---

## 🧠 How It Works

- On page load, the app fetches a random advice slip  
- The advice ID and text are displayed dynamically  
- Clicking the dice button triggers a new API request  
- A spinning animation plays while fetching  
- If something goes wrong, a friendly error message appears
